# Smart Alarm Clock / Réveil intelligent

## English
The alarm clock can run automatically, and uses light and music to wake up more gently.

### What you’ll notice
- Wake-up times can be adjusted automatically.
- Lights can turn on progressively around wake-up time.
- Music can start as part of the wake-up routine.

## Français
Le réveil peut fonctionner automatiquement et utiliser la lumière et la musique pour un réveil plus doux.

### Ce que vous remarquerez
- Les heures de déclenchement peuvent être ajustées automatiquement.
- Les lumières peuvent s’allumer autour de l’heure du réveil.
- La musique peut démarrer dans la routine de réveil.
