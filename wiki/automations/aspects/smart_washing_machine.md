# Smart Washing Machine / Machine à laver

## English
The washing machine has a simple “status” that drives reminders and scheduling.

### What you’ll notice
- The machine can be **scheduled to start at night** (energy-friendly hours).
- You may get a notification when a cycle is **planned**, **finished**, or when the drum should be **emptied**.
- If a **water leak** is detected, power can be cut to reduce risk.

### Status (concept)
- **Inactive** → **Planned** → **Washing in progress** → **To empty** → back to **Inactive**

### Rules working together
- Schedule the cycle at night when requested.
- Keep the machine “status” in sync (planned / running / to empty).
- Notify at key moments (start planned, cycle finished, time to empty).
- Cut power if a leak is detected.

Related: [Home Security & Safety](home_security_and_safety.md), [Voice Commands](voice_commands.md)

## Français
La machine à laver a un “état” simple qui pilote les rappels et la planification.

### Ce que vous remarquerez
- La machine peut être **planifiée la nuit** (heures plus favorables).
- Une notification peut prévenir quand un cycle est **planifié**, **terminé**, ou quand il faut **vider** le tambour.
- En cas de **fuite d’eau**, l’alimentation peut être coupée pour limiter le risque.

### États (principe)
- **Inactive** → **Planifiée** → **Lavage en cours** → **À vider** → retour à **Inactive**

### Règles qui travaillent ensemble
- Planifier le cycle la nuit quand c’est demandé.
- Synchroniser l’état (planifiée / lavage / à vider).
- Notifier aux moments clés (démarrage prévu, cycle terminé, à vider).
- Couper l’alimentation en cas de fuite.
