# Voice Commands / Commandes vocales

## English
Some behaviors can be triggered by voice (phrases are currently in French). This is mostly used for “quick confirmations” and a few routines.

### Common phrases (examples)
- Leaving the home: “Je pars” / “On s’en va” (starts the leaving flow).
- Mark bins as taken out: “J’ai sorti la poubelle”.
- Mark mail as collected: “J’ai récupéré le courrier”.
- Mark litter box as cleaned: “J’ai nettoyé la litière”.
- Reset lights: “Réinitialise les lumières”.
- Office standing desk: “Lève le bureau” / “Baisse le bureau”.
- Music: “Joue …” / “Écoute …” (followed by what you want to play).

### What it’s used for
- Quick confirmations (trash/mail/litter) so you don’t need to open an app.
- A few “house state” shortcuts (like starting the leaving flow).

Related: [Home Presence Modes](home_presence_modes.md), [Trash Bins](smart_trash.md), [Mailbox](smart_mailbox.md), [Litter Box](smart_cat_litter.md)

## Français
Certaines actions peuvent se déclencher à la voix (les phrases sont actuellement en français). C’est surtout utilisé pour des “confirmations rapides” et quelques routines.

### Phrases courantes (exemples)
- Départ : “Je pars” / “On s’en va” (lance le flux de départ).
- Confirmer les poubelles : “J’ai sorti la poubelle”.
- Confirmer le courrier : “J’ai récupéré le courrier”.
- Confirmer la litière : “J’ai nettoyé la litière”.
- Réinitialiser les lumières : “Réinitialise les lumières”.
- Bureau assis/debout : “Lève le bureau” / “Baisse le bureau”.
- Musique : “Joue …” / “Écoute …” (suivi de ce que vous voulez lancer).

### À quoi ça sert
- Des confirmations rapides (poubelles/courrier/litière) sans ouvrir d’application.
- Quelques raccourcis “état de la maison” (comme lancer un départ).
