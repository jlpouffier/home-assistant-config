# Crazy Wake Up / Crazy Wake Up

## English
### Flow (trace-style)
- Entry: when this script is run
- Adjust volume
- Play media
- Wait for: Assistant vocal Lecteur multimédia (Chambre) to “playing”
- Turn on lights in Chambre
- Wait 12s
- Turn off lights in Chambre
- Wait 2s
- REPEAT
  - Turn on Lumières (Chambre)
  - Wait 1s
