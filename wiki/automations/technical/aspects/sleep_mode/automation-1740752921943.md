# Mode Sommeil Child - Actions / Mode Sommeil Enfant - Actions

## English
- Back to guest-friendly view: [sleep_mode](../../../aspects/sleep_mode.md)
- Back to technical aspect index: [sleep_mode](../sleep_mode.md)

### Summary
- Runs when: Mode Sommeil (Chambre de Child)
- Only if: No extra conditions
- Then: Execute the flow below

### Scripts called
- [Démarrer la routine du sommeil de Child](../../scripts/start_child_bedtime_routine.md)

### Flow (trace-style)
- Triggers:
  - Mode Sommeil (Chambre de Child)
- Conditions (all must be true):
  - No extra conditions
- Actions:
  - CHOOSE (first match)
    - Branch 1
      - Check: Mode Sommeil (Chambre de Child) is “on”
      - THEN
        - Turn off lights in Couloir
        - Activate Lecture (Chambre de Child)
        - IF
          - Check: Fenêtre (Chambre de Child) is “off”
          - THEN
            - Close Rideau (Chambre de Child)
        - Run script: Démarrer la routine du sommeil de Child
    - Branch 2
      - Check: Mode Sommeil (Chambre de Child) is “off”
      - THEN
        - Pause Enceintes (Chambre de Child)
        - IF
          - Check: Fenêtre (Chambre de Child) is “off”
          - THEN
            - Open Rideau (Chambre de Child)
