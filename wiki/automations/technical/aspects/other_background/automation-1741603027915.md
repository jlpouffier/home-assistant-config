# Mode Chien - Actions / Mode Chien - Actions

## English
- Back to guest-friendly view: [other_background](../../../aspects/other_background.md)
- Back to technical aspect index: [other_background](../other_background.md)

### Summary
- Runs when: Mode Chien; Modes de presence de la maison; Mode invité; Simulation de présence; Station météo Lumière détectée (Jardin)
- Only if: Modes de presence de la maison is “Vide”; Mode invité is “off”; Simulation de présence is “off”
- Then: Execute the flow below

### Scripts called
- [Éteindre et Sécuriser la maison](../../scripts/leave_home.md)

### Flow (trace-style)
- Triggers:
  - Mode Chien
  - Modes de presence de la maison
  - Mode invité
  - Simulation de présence
  - Station météo Lumière détectée (Jardin)
- Conditions (all must be true):
  - Modes de presence de la maison is “Vide”
  - Mode invité is “off”
  - Simulation de présence is “off”
- Actions:
  - CHOOSE (first match)
    - Branch 1
      - Check: Mode Chien is “on”
      - THEN
        - IF
          - Check: Fenêtre (Bureau) is “off”
          - THEN
            - Open Rideau (Bureau)
        - Turn off lights in Jardin
        - CHOOSE (first match)
          - Branch 1
            - Check: Station météo Lumière détectée (Jardin) is “on”
            - THEN
              - Close Shutter
                - IF
                  - Check: Surveillance de la maison - Fermer le volet quand maison vide is “on”
                  - THEN
                    - Set cover position Volets (Salon)
          - Branch 2
            - Check: Station météo Lumière détectée (Jardin) is “off”
            - THEN
              - Turn on Suspension (Bureau), Cuisine, Salon, Entrée
              - Close Shutter
                - IF
                  - Check: Surveillance de la maison - Fermer le volet quand maison vide is “on”
                  - THEN
                    - Close Volets (Salon)
    - Branch 2
      - Check: Mode Chien is “off”
      - THEN
        - Run script: Éteindre et Sécuriser la maison
