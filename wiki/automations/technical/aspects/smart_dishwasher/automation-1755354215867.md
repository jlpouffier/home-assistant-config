### Summary
- Runs when: État (Cuisine) to “on”
- Only if: No extra conditions
- Then: Execute the flow below

### Flow (trace-style)
- Triggers:
  - État (Cuisine) to “on”
- Conditions (all must be true):
  - No extra conditions
- Actions:
  - Set État lave vaisselle to “Inactif”
  - Make sure door is closed
    - IF
      - Check: Porte (Cuisine) is “open”
      - THEN
        - Wait for: Porte (Cuisine) to “closed”
  - Wait 5m
  - Make sure dishwasher is ready (Not already started)
    - IF
      - Check: État (Cuisine) is “ready”
      - Check: État (Cuisine) is “on”
      - THEN
        - Set État lave vaisselle to “Planifié”
        - Turn off État (Cuisine)
