# Climatisation Intelligente - Chambre Child / Climatisation Intelligente - Chambre Enfant

## English
- Back to guest-friendly view: [smart_air_conditioning](../../../aspects/smart_air_conditioning.md)
- Back to technical aspect index: [smart_air_conditioning](../smart_air_conditioning.md)

### Summary
- Runs when: Climatisation Autorisation (Chambre de Child); Cible chauffage (Chambre de Child); Cible clim (Chambre de Child); Mode climatisation intelligente; Home Assistant starts; Climatisation intelligente (Chambre de Child)
- Only if: Climatisation intelligente (Chambre de Child) is “on”
- Then: Execute the flow below

### Flow (trace-style)
- Triggers:
  - Climatisation Autorisation (Chambre de Child)
  - Cible chauffage (Chambre de Child)
  - Cible clim (Chambre de Child)
  - Mode climatisation intelligente
  - Home Assistant starts
  - Climatisation intelligente (Chambre de Child)
- Conditions (all must be true):
  - Climatisation intelligente (Chambre de Child) is “on”
- Actions:
  - Wait 1s
  - IF
    - Check: Climatisation Autorisation (Chambre de Child) is “on”
    - THEN
      - CHOOSE (first match)
        - Branch 1
          - Check: Mode climatisation intelligente is “Froid”
          - THEN
            - IF
              - Check: NOT (Climatisation (Chambre de Child) is “cool”)
              - THEN
                - Set Climatisation (Chambre de Child) mode to Cooling
                - Wait 1s
            - IF
              - Check: A custom rule is true (based on: Cible clim (Chambre de Child), Climatisation (Chambre de Child))
              - THEN
                - Set Climatisation (Chambre de Child) temperature to the target temperature
        - Branch 2
          - Check: Mode climatisation intelligente is “Chaud”
          - THEN
            - IF
              - Check: NOT (Climatisation (Chambre de Child) is “heat”)
              - THEN
                - Set Climatisation (Chambre de Child) mode to Heating
                - Wait 1s
            - IF
              - Check: A custom rule is true (based on: Cible chauffage (Chambre de Child), Climatisation (Chambre de Child))
              - THEN
                - Set Climatisation (Chambre de Child) temperature to the target temperature
    - ELSE
      - IF
        - Check: NOT (Climatisation (Chambre de Child) is “off”)
        - THEN
          - Set Climatisation (Chambre de Child) mode to Off

      - CHOISIR (première branche valide)

              - ALORS
                - Régler Climatisation (Chambre de Enfant) sur la température cible
        - Branche 2
          - Vérifier : Mode climatisation intelligente est “Chaud”
          - ALORS
            - SI
              - Vérifier : NON (Climatisation (Chambre de Enfant) est “heat”)
              - ALORS
                - Mettre Climatisation (Chambre de Enfant) en mode Chaud
                - Attendre 1s
            - SI
              - Vérifier : Une règle personnalisée est vraie (basée sur : Cible chauffage (Chambre de Enfant), Climatisation (Chambre de Enfant))
              - ALORS
                - Régler Climatisation (Chambre de Enfant) sur la température cible
    - SINON
      - SI
        - Vérifier : NON (Climatisation (Chambre de Enfant) est “off”)
        - ALORS
          - Mettre Climatisation (Chambre de Enfant) en mode Éteint
