# Climatisation Intelligente - Chambre / Climatisation Intelligente - Chambre

## English
- Back to guest-friendly view: [smart_air_conditioning](../../../aspects/smart_air_conditioning.md)
- Back to technical aspect index: [smart_air_conditioning](../smart_air_conditioning.md)

### Summary
- Runs when: Climatisation Autorisation (Chambre); Cible chauffage (Chambre); Cible clim (Chambre); Mode climatisation intelligente; Home Assistant starts; Climatisation intelligente (Chambre)
- Only if: Climatisation intelligente (Chambre) is “on”
- Then: Execute the flow below

### Flow (trace-style)
- Triggers:
  - Climatisation Autorisation (Chambre)
  - Cible chauffage (Chambre)
  - Cible clim (Chambre)
  - Mode climatisation intelligente
  - Home Assistant starts
  - Climatisation intelligente (Chambre)
- Conditions (all must be true):
  - Climatisation intelligente (Chambre) is “on”
- Actions:
  - Wait 1s
  - IF
    - Check: Climatisation Autorisation (Chambre) is “on”
    - THEN
      - CHOOSE (first match)
        - Branch 1
          - Check: Mode climatisation intelligente is “Froid”
          - THEN
            - IF
              - Check: NOT (Climatisation (Chambre) is “cool”)
              - THEN
                - Set Climatisation (Chambre) mode to Cooling
                - Wait 1s
            - IF
              - Check: A custom rule is true (based on: Cible clim (Chambre), Climatisation (Chambre))
              - THEN
                - Set Climatisation (Chambre) temperature to the target temperature
        - Branch 2
          - Check: Mode climatisation intelligente is “Chaud”
          - THEN
            - IF
              - Check: NOT (Climatisation (Chambre) is “heat”)
              - THEN
                - Set Climatisation (Chambre) mode to Heating
                - Wait 1s
            - IF
              - Check: A custom rule is true (based on: Cible chauffage (Chambre), Climatisation (Chambre))
              - THEN
                - Set Climatisation (Chambre) temperature to the target temperature
    - ELSE
      - IF
        - Check: NOT (Climatisation (Chambre) is “off”)
        - THEN
          - Set Climatisation (Chambre) mode to Off

        - Branche 2
          - Vérifier : Mode climatisation intelligente est “Chaud”
          - ALORS
            - SI
              - Vérifier : NON (Climatisation (Chambre) est “heat”)
              - ALORS
                - Mettre Climatisation (Chambre) en mode Chaud
                - Attendre 1s
            - SI
              - Vérifier : Une règle personnalisée est vraie (basée sur : Cible chauffage (Chambre), Climatisation (Chambre))
              - ALORS
                - Régler Climatisation (Chambre) sur la température cible
    - SINON
      - SI
        - Vérifier : NON (Climatisation (Chambre) est “off”)

          - Mettre Climatisation (Chambre) en mode Éteint
