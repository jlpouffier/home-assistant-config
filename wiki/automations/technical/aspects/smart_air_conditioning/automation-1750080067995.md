# Climatisation Intelligente - Bureau / Climatisation Intelligente - Bureau

## English
- Back to guest-friendly view: [smart_air_conditioning](../../../aspects/smart_air_conditioning.md)

### Summary
- Runs when: Climatisation Autorisation (Bureau); Cible chauffage (Bureau); Cible clim (Bureau); Mode climatisation intelligente; Home Assistant starts; Climatisation intelligente (Bureau)
- Only if: Climatisation intelligente (Bureau) is “on”
- Then: Execute the flow below

### Flow (trace-style)
- Triggers:
  - Climatisation Autorisation (Bureau)
  - Cible chauffage (Bureau)
  - Cible clim (Bureau)
  - Mode climatisation intelligente
  - Home Assistant starts
  - Climatisation intelligente (Bureau)
- Conditions (all must be true):
  - Climatisation intelligente (Bureau) is “on”
- Actions:
  - Wait 1s
  - IF
    - Check: Climatisation Autorisation (Bureau) is “on”
    - THEN
      - CHOOSE (first match)
        - Branch 1
          - Check: Mode climatisation intelligente is “Froid”
          - THEN
            - IF
              - Check: NOT (Climatisation (Bureau) is “cool”)
              - THEN
                - Set Climatisation (Bureau) mode to Cooling
                - Wait 1s
            - IF
              - Check: A custom rule is true (based on: Cible clim (Bureau), Climatisation (Bureau))
              - THEN
                - Set Climatisation (Bureau) temperature to the target temperature
        - Branch 2
          - Check: Mode climatisation intelligente is “Chaud”
          - THEN
            - IF
              - Check: NOT (Climatisation (Bureau) is “heat”)
              - THEN
                - Set Climatisation (Bureau) mode to Heating
                - Wait 1s
            - IF
              - Check: A custom rule is true (based on: Cible chauffage (Bureau), Climatisation (Bureau))
              - THEN
                - Set Climatisation (Bureau) temperature to the target temperature
    - ELSE
      - IF
        - Check: NOT (Climatisation (Bureau) is “off”)
        - THEN
          - Set Climatisation (Bureau) mode to Off

            - SI
