### Summary
- Runs when: Modes de presence de la maison to “Vide”; Modes de presence de la maison from “Vide”; Ratio de présence rez-de-chaussée; Ouvertures Pièce de vie (Salon) for 5m; Home Assistant starts; Mode invité; Mode Sommeil (Chambre); Mode Cinema (Salon)
- Only if: No extra conditions
- Then: Execute the flow below

### Flow (trace-style)
- Triggers:
  - Modes de presence de la maison to “Vide”
  - Modes de presence de la maison from “Vide”
  - Ratio de présence rez-de-chaussée
  - Ouvertures Pièce de vie (Salon) for 5m
  - Home Assistant starts
  - Mode invité
  - Mode Sommeil (Chambre)
  - Mode Cinema (Salon)
- Conditions (all must be true):
  - No extra conditions
- Actions:
  - CHOOSE (first match)
    - Branch 1
      - Check: Ouvertures Pièce de vie (Salon) is “on”
      - THEN
        - Set Mode chauffage/clim (Salon) to “Frost Guard”
    - Branch 2
      - Check: (NOT (Modes de presence de la maison is “Vide”) OR Mode invité is “on”)
      - Check: (Ratio de présence rez-de-chaussée > 50 OR Mode Cinema (Salon) is “on”)
      - Check: Mode Sommeil (Chambre) is “off”
      - THEN
        - Set Mode chauffage/clim (Salon) to “Présent”
    - Default
      - Set Mode chauffage/clim (Salon) to “Absent”
