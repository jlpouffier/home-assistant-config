### Summary
- Runs when: Modes de presence de la maison to “Vide”; Modes de presence de la maison from “Vide”; Fenêtre (Chambre) for 5m; Capteur de présence Ratio de présence (Chambre); Home Assistant starts; Mode invité; Mode Sommeil (Chambre)
- Only if: No extra conditions
- Then: Execute the flow below

### Flow (trace-style)
- Triggers:
  - Modes de presence de la maison to “Vide”
  - Modes de presence de la maison from “Vide”
  - Fenêtre (Chambre) for 5m
  - Capteur de présence Ratio de présence (Chambre)
  - Home Assistant starts
  - Mode invité
  - Mode Sommeil (Chambre)
- Conditions (all must be true):
  - No extra conditions
- Actions:
  - CHOOSE (first match)
    - Branch 1
      - Check: Fenêtre (Chambre) is “on”
      - THEN
        - Set Mode chauffage/clim (Chambre) to “Frost Guard”
    - Branch 2
      - Check: (NOT (Modes de presence de la maison is “Vide”) OR Mode invité is “on”)
      - Check: (Capteur de présence Ratio de présence (Chambre) > 50 OR Mode Sommeil (Chambre) is “on”)
      - THEN
        - Set Mode chauffage/clim (Chambre) to “Présent”
    - Default
      - Set Mode chauffage/clim (Chambre) to “Absent”
