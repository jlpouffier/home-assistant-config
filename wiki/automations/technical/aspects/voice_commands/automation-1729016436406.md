### Summary
- Runs when: Corvées; Modes de presence de la maison; Mode Cinema (Salon); Mode Sommeil (Chambre); Home Assistant starts
- Only if: No extra conditions
- Then: Execute the flow below

### Flow (trace-style)
- Triggers:
  - Corvées
  - Modes de presence de la maison
  - Mode Cinema (Salon)
  - Mode Sommeil (Chambre)
  - Home Assistant starts
- Conditions (all must be true):
  - No extra conditions
- Actions:
  - IF
    - Check: Modes de presence de la maison is “Occupée”
    - Check: Mode Cinema (Salon) is “off”
    - Check: Mode Sommeil (Chambre) is “off”
    - THEN
      - IF
        - Check: Corvées is “on”
        - THEN
          - Turn on Assistant vocal LEDs (Salon)
        - ELSE
          - Turn on Assistant vocal LEDs (Salon)
    - ELSE
      - Turn off Assistant vocal LEDs (Salon)
