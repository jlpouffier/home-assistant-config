# Simulation de présence / Simulation de présence

## English
- Back to guest-friendly view: [presence_simulation](../../../aspects/presence_simulation.md)
- Back to technical aspect index: [presence_simulation](../presence_simulation.md)

### Summary
- Runs when: A trigger fires; Simulation de présence; Simulation de présence - Contrôle des ouvertures; Modes de presence de la maison; Home Assistant starts
- Only if: Simulation de présence is “on”; Modes de presence de la maison is “Vide”; Mode invité is “off”
- Then: Execute the flow below

### Flow (trace-style)
- Triggers:
  - A trigger fires
  - Simulation de présence
  - Simulation de présence - Contrôle des ouvertures
  - Modes de presence de la maison
  - Home Assistant starts
- Conditions (all must be true):
  - Simulation de présence is “on”
  - Modes de presence de la maison is “Vide”
  - Mode invité is “off”
- Actions:
  - Get events
  - IF
    - Check: Porte d'entrée (Entrée) is “unlocked”
    - THEN
      - Wait for: Porte d'entrée (Entrée) from “locked” (timeout 30m)
  - CHOOSE (first match)
    - Branch 1
      - Check: A custom rule is true
      - THEN
        - Turn off lights in Jardin
        - IF
          - Check: Simulation de présence - Contrôle des ouvertures is “on”
          - THEN
            - Close Volets (Salon)
            - Close Rideaux Premier Étage
    - Branch 2
      - Check: A custom rule is true
      - THEN
        - Turn off lights in Jardin, Couloir
        - Turn on lights in Bureau, Chambre, Chambre de Child
        - IF
          - Check: Simulation de présence - Contrôle des ouvertures is “on”
          - THEN
            - Close Volets (Salon)
            - Open Rideaux Premier Étage
    - Branch 3
      - Check: A custom rule is true
      - THEN
        - Turn off lights in Jardin
        - Turn on Sphère (Jardin), Applique (Jardin), Cuisine, Entrée, Salon
        - IF
          - Check: Simulation de présence - Contrôle des ouvertures is “on”
          - THEN
            - Open Volets (Salon)
            - Open Rideaux Premier Étage
    - Branch 4
      - Check: A custom rule is true
      - THEN
        - Turn off a target
        - Turn on Guirlande (Jardin), Cuisine, Entrée, Salon
        - IF
          - Check: Simulation de présence - Contrôle des ouvertures is “on”
          - THEN
            - Open Volets (Salon)
            - Open Rideaux Premier Étage
    - Branch 5
      - Check: A custom rule is true
      - THEN
        - Turn off a target
        - Turn on Sphère (Jardin), Applique (Jardin), Cuisine, Entrée, Salon
        - IF
          - Check: Simulation de présence - Contrôle des ouvertures is “on”
          - THEN
            - Open Volets (Salon)
            - Open Rideaux Premier Étage
    - Branch 6
      - Check: A custom rule is true
      - THEN
        - Turn off lights in Couloir, Jardin
        - Turn on lights in Bureau, Chambre, Chambre de Child
        - IF
          - Check: Simulation de présence - Contrôle des ouvertures is “on”
          - THEN
            - Open Volets (Salon)
            - Open Rideaux Premier Étage
