### Summary
- Runs when: Mode Cinema (Salon) to “on”
- Only if: TV intelligente is “on”; Mode invité is “off”; Modes de presence de la maison is “Occupée”; Présence Rez-de-chaussée is “on”
- Then: Execute the flow below

### Flow (trace-style)
- Triggers:
  - Mode Cinema (Salon) to “on”
- Conditions (all must be true):
  - TV intelligente is “on”
  - Mode invité is “off”
  - Modes de presence de la maison is “Occupée”
  - Présence Rez-de-chaussée is “on”
- Actions:
  - Create a temporary scene snapshot
  - Turn off Applique escalier (Couloir), Jardin, Cuisine, Entrée, Salon
  - Activate Salon TV Playing (Salon)
  - IF
    - Check: Station météo Lumière détectée (Jardin) is “on”
    - THEN
      - Close Volets (Salon)
  - IF
    - Check: Mode Cinema (Salon) is “on”
    - THEN
      - Wait for: Mode Cinema (Salon) to “off”
  - Activate Dynamic Scene Before Movie Mode
  - Delete Dynamic Scene Before Movie Mode
