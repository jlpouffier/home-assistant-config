# Home Automations Guide / Guide des automatisations

## English
This wiki explains how the home behaves day-to-day (comfort, chores, safety), in a way that should make sense even if you don’t know how it’s built.

> Note: This wiki is AI-generated from the Home Assistant configuration and is a work in progress. Expect imperfections.

- This guide is organized by **aspects** (Smart Heating, Smart Trash, Sleep Mode, …).
- It focuses on **what you experience** (what happens, when it happens, what you might want to do).
- Names are **human-friendly** (no technical IDs); personal names are anonymized except **JLo**.

- Want the nerdy details? See [Technical Automations](technical/README.md).
